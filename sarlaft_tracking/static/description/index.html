SARLAFT Tracking Module

This module provides a structured and traceable system for recording SARLAFT (Sistema de Administración del Riesgo de Lavado de Activos y de la Financiación del Terrorismo) queries performed on customers, suppliers, and employees.

Key Features:

  • Register and track SARLAFT queries
  • Mass import from Excel and CSV files
  • Automated notifications for upcoming queries
  • Integration with contacts and employees
  • Role-based access control
