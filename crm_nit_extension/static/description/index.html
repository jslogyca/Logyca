CRM NIT Extension

Módulo de extensión para Odoo 17 CRM que agrega el campo NIT (Número de Identificación Tributaria) para registros de empresas en leads.

Características Principales

  • Campo NIT para identificación tributaria de empresas
  • Visible únicamente en registros de tipo Lead
  • Tracking automático de cambios
  • Integración nativa con CRM de Odoo
  • Compatible con Odoo 17

Funcionalidad

El campo NIT aparece automáticamente en el formulario de Lead, ubicado después del campo "Nombre de la empresa" (partner_name).

Visibilidad Condicional:
El campo solo se muestra cuando:
  • type = 'lead' (es un Lead)
  • type = False (registro nuevo sin tipo definido)

Caso de Uso

Este módulo es especialmente útil para empresas colombianas que necesitan registrar el NIT de sus clientes potenciales durante la etapa de Lead, facilitando la identificación tributaria antes de convertir el Lead en Oportunidad.

Instalación

  1. Copiar el módulo a la carpeta de addons de Odoo
  2. Actualizar la lista de aplicaciones en Odoo
  3. Buscar e instalar "CRM NIT Extension"
